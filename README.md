es-playground
=============

Simple Node.js + @babel/preset-env playground that allows to test latest ECMAScript features

###### [Install](#Install) | [Example](#Example) | [License](#License)

Install
-------

```bash
$ git clone --depth=1 https://github.com/eu-ge-ne/es-playground.git
$ cd es-playground
$ npm install
```

Example
-------

```bash
$ npm start
```
